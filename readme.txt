== Description ==

Permettez à vos visiteurs de voter pour vos articles sur Ôdélices. Ajoute un bouton "J'aime cet article" à la fin de vos articles.

== Installation ==

Télécharger l'extension `communaute-o-delices` et déposer les fichiers dans le dossier `/wp-content/plugins/`. 
Activer l'extension : c'est terminé !
